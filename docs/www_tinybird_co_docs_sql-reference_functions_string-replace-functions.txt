overlay
В¶
Replaces a segment of an input string with another string, starting at a specified 1-based offset. You can optionally define the length of the segment to be replaced.
Syntax
В¶
overlay(s, replace, offset[, length])
Arguments
В¶
s
:
String
. The original string.
replace
:
String
. The string to insert.
offset
:
Int
. The 1-based starting position for the replacement. A negative offset counts from the end of the string.
length
: Optional.
Int
. The number of bytes in
s
to replace. If omitted, the length of
replace
is used.
Returns
В¶
A
String
with the specified segment replaced.
Example
В¶
SELECT overlay('My father is from Mexico.', 'mother', 4) AS res

overlayUTF8
В¶
Replaces a segment of an input string with another string, similar to
overlay
, but operates on UTF-8 characters rather than bytes. It assumes the input string is valid UTF-8.
Syntax
В¶
overlayUTF8(s, replace, offset[, length])
Arguments
В¶
s
:
String
. The original UTF-8 string.
replace
:
String
. The UTF-8 string to insert.
offset
:
Int
. The 1-based starting character position for the replacement. A negative offset counts from the end of the string.
length
: Optional.
Int
. The number of UTF-8 characters in
s
to replace. If omitted, the character length of
replace
is used.
Returns
В¶
A
String
with the specified segment replaced, preserving UTF-8 encoding.

replaceRegexpOne
В¶
Replaces the first substring that matches a given regular expression
pattern
with a specified
replacement
string. The
replacement
string can include backreferences to capturing groups from the
pattern
.
Syntax
В¶
replaceRegexpOne(haystack, pattern, replacement)
Arguments
В¶
haystack
:
String
. The input string to search within.
pattern
:
String
. The regular expression (RE2 syntax) to match.
replacement
:
String
. The string to insert. Can contain
\0
for the entire match or
\1
through
\9
for capturing groups.
Returns
В¶
A
String
with the first regex match replaced.
Example
В¶

replaceRegexpAll
В¶
Replaces all substrings that match a given regular expression
pattern
with a specified
replacement
string. This function behaves like
replaceRegexpOne
but applies the replacement globally.
Alias:
REGEXP_REPLACE
.
Syntax
В¶
replaceRegexpAll(haystack, pattern, replacement)
Arguments
В¶
haystack
:
String
. The input string to search within.
pattern
:
String
. The regular expression (RE2 syntax) to match.
replacement
:
String
. The string to insert. Can contain
\0
for the entire match or
\1
through
\9
for capturing groups.
Returns
В¶
A
String
with all regex matches replaced.

В¶

translate
В¶
Replaces characters in an input string based on a one-to-one mapping defined by two other strings,
from
and
to
. Each character in the input string that matches a character in
from
is replaced by the character at the corresponding position in
to
. If
from
contains characters not present in
to
, those characters are deleted from the input string. This function operates on ASCII characters; non-ASCII characters are not modified.
Syntax
В¶
translate(s, from, to)
Arguments
В¶
s
:
String
. The input string.
from
:
String
. A string containing characters to be replaced.
to
:
String
. A string containing replacement characters.
Returns
В¶
A
String
with characters replaced or deleted according to the mapping.
Example
В¶
SELECT translate('Hello, World!', 'delor', 'DELOR') AS res
Result:
в”Ңв”Җresв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ HELLO, WORLD! в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
from
and
to
arguments have different lengths:
SELECT translate('clickhouse', 'clickhouse', 'CLICK') AS res
Result:
в”Ңв”Җresв”Җв”Җв”Җв”җ
в”Ӯ CLICK в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

В¶

%d

%

Syntax
В¶
printf(format, arg1, arg2, ...)
Arguments
В¶
format
:
String
. The format string containing specifiers.
arg1, arg2, ...
: Any type. The values to be formatted into the string.
Returns
В¶
A
String
with the arguments formatted according to the specifiers.
