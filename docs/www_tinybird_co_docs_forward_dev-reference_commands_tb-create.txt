tb create (deprecated)
¶
Copy as MD
tb create
is deprecated.
Use
tb init
to scaffold projects.
The command is kept only for backward compatibility and prints a deprecation warning.
Replacement
¶
Use
tb init
instead:
tb init --type cli
Common equivalents:
Choose project folder:
tb init --folder tinybird
Set development mode:
tb init --dev-mode manual
Generate CI/CD templates:
tb init --cicd github
or
tb init --cicd gitlab
