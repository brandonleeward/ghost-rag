Export data from Tinybird to Amazon S3
Learn how to use Tinybird's S3 Sink to export the results of your Tinybird Pipes to Amazon S3. We will provide guidance and best practices on how and when to use the S3 Sink, including information on observability, pricing, and optimizations.
In this free product training, you'll learn...
How to export the results of a Tinybird Pipe into Amazon S3
How to set up and monitor S3 Sink Jobs in Tinybird
How to set up secure access policies from Tinybird to Amazon S3
How to use file template syntax to define how files are created in S3
Performance, pricing, and limitations of the S3 Sink
