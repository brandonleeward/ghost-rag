tb connection
¶
Copy as MD
Manage your data connections. Global options apply to this command. See
Global options
.
Connections are configured through .connection files. See
Connection files
.
Subcommands
¶
The following subcommands are available:
Subcommand
Description
create TYPE
Create a new connection. Supported values are:
kafka
,
gcs
, and
s3
.
ls
Lists all connections in the project.
tb connection ls
¶
Lists connections.
Option
Description
--service TEXT
Filter by service.
