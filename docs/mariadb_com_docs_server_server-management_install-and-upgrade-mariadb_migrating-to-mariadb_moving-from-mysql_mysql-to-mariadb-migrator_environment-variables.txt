Environment Variables
The environment variables required to run each MySQL to MariaDB Migrator mode, plus the common source, target, and optional tuning variables.
Enterprise tooling.
The MySQL to MariaDB Migrator is proprietary MariaDB software, available to MariaDB customers and partners under approved usage terms. It is not open source and is not available for general public use.
The migrator reads its configuration from environment variables (or the configuration file it writes from your interactive answers). This page lists the variables
required
to run each mode. A complete reference of every variable, with defaults and descriptions, ships with the tool.
The interactive launcher prompts for any required input that is not already set, so you do not have to export these variables by hand. Set them explicitly for scripted or unattended runs.
