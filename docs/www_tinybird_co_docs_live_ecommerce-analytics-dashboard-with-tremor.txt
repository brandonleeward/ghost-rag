Easily build an eCommerce analytics dashboard in minutes
Building an eCommerce site? If you want it to succeed, you'll need to measure everything. Come learn how to easily build a beautiful and responsive eCommerce analytics dashboard from scratch in under an hour.
In this live coding session, you'll learn how to...
Generate mock eCommerce event data and send it to Tinybird using
Mockingbird
, a free mock data stream generator
Analyze your eCommerce data and define key performance indicators using SQL
Publish your eCommerce metrics as parameterized and documented APIs to share and embed in applications
Quickly build beautiful and responsive eCommerce analytics dashboards using the Tremor components library
