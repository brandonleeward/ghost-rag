Creating Mroonga User-Defined Functions
Instructions on how to manually create Mroonga's UDFs if they were not automatically installed, ensuring full functionality.
The
Mroonga storage engine
includes a number of
user-defined functions
that need to be created before they can be used. If these are not created already during Mroonga setup, you will need to do so yourself. The full list of available functions and the statements to create them are found in
share/mroonga/install.sql
, for example, as of Mroonga 7.07 running on Linux:
Copy
DROP
FUNCTION
IF
EXISTS
last_insert_grn_id;
CREATE
FUNCTION
last_insert_grn_id
RETURNS
INTEGER
SONAME
'
ha_mroonga.so
'
;
