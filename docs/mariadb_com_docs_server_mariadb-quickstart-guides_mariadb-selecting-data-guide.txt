This guide explains the SELECT statement in detail, covering how to retrieve, filter, limit, and sort data from your MariaDB database.

SELECT Statement Modifiers
Place these modifiers immediately after the
SELECT
keyword.
ALL
vs
DISTINCT
:
ALL
(default): Returns all rows that meet the criteria.
DISTINCT
: Returns only unique rows for the selected columns. If multiple identical rows are found for the specified columns, only the first one is displayed.

HIGH_PRIORITY

Gives the
