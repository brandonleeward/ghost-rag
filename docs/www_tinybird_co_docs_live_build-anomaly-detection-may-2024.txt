Free Workshop: Building Real-Time Anomaly Detection Systems
In this live coding session, we'll build a real-time anomaly detection system. We'll explore five key methods to identify unusual patterns in your data streams as they occur. By the end, you'll be equipped to implement these techniques and gain valuable insights from your data.
In this free technical training, we will build the following anomaly detection methods:
Timeout thresholds
Rate-of-change checks
Out-of-range boundaries
Generating Z-scores
Calculating interquartile ranges (IQRs)
