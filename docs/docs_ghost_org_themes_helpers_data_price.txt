price
Usage:
{{price plan}}
The
{{price}}
helper formats monetary values from their smallest denomination to a human readable denomination with currency formatting. Example:
{{
price
plan
}}
This will output
$5
.
The
{{price}}
helper accepts a number of optional attributes:
currency
- defaults to
plan.currency
when passed a
plan
object
locale
- defaults to
@site.locale
numberFormat
- defaults to “short”, and can be either “short” ($5) or “long” ($5.00)
currencyFormat
- defaults to “symbol” and can be one of “symbol” ($5), “code” (EUR 5) or “name” (5 euros)
{{price}}
can be used with static values as well,
{{price 4200}}
will output
42
.
The default behaviour of the
price
helper is the same as:
{{
price
plan.amount
currency=plan.currency
locale=
@
site.locale
numberFormat=
"short"
currencyFormat=
"symbol"
}}
Passing a
currency
without a price will output the symbol for that currency:
{{
price
currency=
"USD"
}}
<!-- Outputs: $ -->
​
