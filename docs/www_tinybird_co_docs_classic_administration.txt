Administration
¶
Copy as MD
Effective administration of your Tinybird account involves managing several key components:
Organizations
: Create and manage organizations to group workspaces and team members.
Workspaces
: Set up isolated environments for your data projects.
Auth Tokens
: Generate and control access tokens for secure API authentication.
