POST
/
admin
/
newsletters
/
Required fields:
name
Options:
opt_in_existing
When
opt_in_existing
is set to
true
, existing members with a subscription to one or more active newsletters are also subscribed to this newsletter. The response metadata will include the number of members opted-in.
