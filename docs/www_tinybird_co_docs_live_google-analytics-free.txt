Build a GDPR-compliant alternative to Google Analytics
Analytics on the web is changing. The shift to a "post cookie" world demands new ways to understand how your customers behave on your website. New regulations have even made Google Analytics illegal in some countries. In this livestream, learn how to build a GDPR-compliant Google Analytics alternative with a few lines of SQL and an open-source JavaScript tracker.
During this livestream, we'll show you how to:
Install an open-source JavaScript tracker that streams web events to Tinybird without relying on third-party cookies.
Personalize the JavaScript code to your domain with custom event types.
Implement functionality to mask PII to ensure visitor privacy.
Use SQL to transform events data into useful metrics using Tinybird.
Recreate some common Google Analytics views in a frontend using Tinybird APIs.
