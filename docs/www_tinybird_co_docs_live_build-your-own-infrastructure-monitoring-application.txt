Build your own infrastructure monitoring application
Tune in for a unique learning experience with Joe Karlsson, seasoned Developer Advocate at Tinybird, as he teaches you how to build a miniature version of Datadog's infrastructure monitoring tool. This is an exclusive opportunity for developers seeking to level up their skills, get hands-on with Tinybird, and delve deep into the workings of Kafka.
In this live coding session, you'll learn how to:
Understand the core components of an infrastructure monitoring tool and the core features of a platform such as Datadog
Use Kafka to capture and stream critical infrastructure events
Ingest events into an OLAP datastore using Tinybird's native Kafka connector
Query your data for relevant infrastructure health metrics
Publish and visualize your metrics in a functional infrastructure monitoring tool
