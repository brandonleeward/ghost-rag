Recent CLI versions removed AI-assisted command generation and introduced
tb init
for project setup.
tb init
replaces
tb create
for scaffolding projects.
tb mock
has been removed. Create fixture files manually (or asking a codign agent) in
fixtures/
.
tb test create
has been removed. Create YAML tests manually (or asking a coding agent), then use
tb test run
and
tb test update
.
tb --prompt
and interactive
tb
mode are no longer available.
