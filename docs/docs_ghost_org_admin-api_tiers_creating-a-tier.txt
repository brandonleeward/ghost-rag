Creating a Tier
POST
/
admin
/
tiers
/
Required fields:
name
Create public and hidden tiers by using this endpoint. New tiers are always set as
active
when created.
The example below creates a paid Tier with all properties including custom monthly/yearly prices, description, benefits, and welcome page.
// POST /admin/tiers/
{
"tiers"
: [
{
"name"
:
"Platinum"
,
"description"
:
"Access to everything"
,
"welcome_page_url"
:
"/welcome-to-platinum"
,
"visibility"
:
"public"
,
"monthly_price"
:
1000
,
"yearly_price"
:
10000
,
"currency"
:
"usd"
,
"benefits"
: [
"Benefit 1"
,
"Benefit 2"
]
}
]
}
