tb job
¶
Copy as MD
Manage jobs. Global options apply to this command. See
Global options
.
The following subcommands are available:
Subcommand
Description
cancel JOB_ID
Tries to cancel a job.
details JOB_ID
Gets details for any job created in the last 48h.
ls [OPTIONS]
Lists jobs. Use
--status [waiting|working|done|error]
or
-s
to show results with the desired status.
