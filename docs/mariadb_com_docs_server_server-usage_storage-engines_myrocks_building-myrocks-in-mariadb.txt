Building MyRocks in MariaDB
Learn how to build the MyRocks storage engine from source within the MariaDB Server build process, including necessary dependencies and configuration options.
This page describes how to get
MyRocks in MariaDB
when compiling MariaDB from source. (See
Build-Steps
for instructions how to build the upstream.)
Build Process and Requirements
MariaDB compile process will compile
MyRocks
into
ha_rocksdb.so
by default if the platform supports it (That is, no WITH_ROCKSDB switch is necessary).
Platform requirements:
A 64-bit platform (due to some 32 bit compilers having difficulties with RocksDB)
git installed (or git submodules fetched somehow)
A sufficiently recent compiler:
gcc >= 4.8, or
clang >= 3.3, or
MS Visual Studio 2015 or newer
