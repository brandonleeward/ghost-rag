Checking and Repairing CSV Tables
Learn how to use CHECK TABLE and REPAIR TABLE to identify and fix corruptions in CSV tables, discarding rows from the first error onwards.
CSV tables
support the
CHECK TABLE
and
REPAIR TABLE
statements.
CHECK TABLE
marks the table as corrupt if it finds a problem, while
REPAIR TABLE
restores rows until the first corrupted row, discarding the rest.
