TokuDB Resources
The TokuDB storage engine has been removed from MariaDB.
TokuDB has been deprecated by its upstream maintainer. It is disabled from MariaDB 10.5 and has been removed in MariaDB 10.6 -
MDEV-19780
. We recommend
MyRocks
as a long-term migration path.
This page contains links to various online resources for
TokuDB
.
Website
Tokutek was the primary developer of TokuDB before being acquired by Percona.
Their website
.
Percona's TokuDB Documentation
TokuDB Documentation
Mailing Lists
There were two TokuDB specific mailing lists run by Tokutek:
tokudb-user
- a mailing list for users of TokuDB
tokudb-dev
- a mailing list TokuDB contributors
These were discontinued and redirected to
Percona's MySQL and MariaDB forum
.
IRC

#tokutek

Freenode
.

#tokutek
channel using Freenode's web IRC client:
?randomnick=1&channels=tokutek&prompt=1
