Real-time filtering and aggregation with DynamoDB
A live coding session for developers looking to do more with their DynamoDB data.
We'll use a real-world example of an airline loyalty program, demonstrating how to track customer miles, spending, and reward progress in real-time.
Along with learning about when and why to use DynamoDB you'll learn how to:
Set up and seed a DynamoDB table
Ingest data from DynamoDB into Tinybird
Build SQL queries that can dynamically filter and aggregate
Publish SQL queries as REST APIs to call from the demo app
Create visualizations with your REST APIs
By the end of this session, you'll have the knowledge and skills to build your own real-time dashboards with DynamoDB and Tinybird.
