Partitions Metadata
Understand how to retrieve metadata about partitions using the INFORMATION_SCHEMA.PARTITIONS table to monitor row counts and storage usage.
The
PARTITIONS
table in the
INFORMATION_SCHEMA
database contains information about partitions.
The
SHOW TABLE STATUS
statement contains a
Create_options
column, that contains the string 'partitioned' for partitioned tables.
The
SHOW CREATE TABLE
statement returns the
CREATE TABLE
statement that can be used to re-create a table, including the partitions definition.
