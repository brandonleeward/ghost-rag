Nothing
¶
Copy as MD
This data type isn't supported at ingest. It is only supported at query time and to create Copy Data Sources or Materialized View Data Sources.
The only purpose of this data type is to represent cases where a value isn't expected. So you can’t create a
Nothing
type value.
For example, literal NULL has type of
Nullable(Nothing)
.
The
Nothing
type can also used to denote empty arrays:
SELECT toTypeName(array())
┌─toTypeName(array())─┐
│ Array(Nothing)      │
└─────────────────────┘
