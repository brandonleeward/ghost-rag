tb open
¶
Copy as MD
Opens a workspace in Tinybird Cloud. You can get your current workspace by running
tb workspace current
. Global options apply to this command. See
Global options
.
For example:
# Opens the current workspace
tb open

# Opens a specific workspace in the cloud environment
tb --cloud open --workspace someworkspace
The following options are available:
Option
Description
--workspace
Sets the workspace you want to open. If unset, your current workspace is used.
