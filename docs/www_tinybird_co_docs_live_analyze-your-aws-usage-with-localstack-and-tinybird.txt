Analyze your AWS usage with LocalStack and Tinybird
Is your team developing cloud applications for AWS? Want to see how your team is using AWS APIs? With LocalStack and Tinybird, you can get those insights in real time! In this livestream, you will learn how to write a LocalStack extension to record your AWS API calls from into Tinybird. Then, using Tinybird, create APIs to track usage analytics in realtime.
During this livestream, we'll show you how to:
Use LocalStack to run your AWS application locally.
Write a LocalStack extension that records AWS API calls made by your application.
Log AWS API calls into Tinybird using Python and the Tinybird Events API.
Create materialized views and write queries to get insights from all your LocalStack instances.
