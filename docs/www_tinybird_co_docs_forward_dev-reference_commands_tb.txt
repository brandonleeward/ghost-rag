tb
¶
Copy as MD
tb
is the root command for the Tinybird CLI.
Usage
¶
Run
tb
without a subcommand to see help:
tb
Use subcommands for specific actions, for example:
tb init
tb build
tb deploy
tb test run
tb --prompt
and interactive AI mode are no longer available.
Common root options
¶
Global options are passed before the subcommand. For example:
tb --cloud deploy
tb --branch=my_feature build
tb --output json deployment ls
Examples
¶
tb --help
tb --version
tb init --type cli --dev-mode manual
tb --cloud deploy
For the full list of global flags, see
Global options
.
