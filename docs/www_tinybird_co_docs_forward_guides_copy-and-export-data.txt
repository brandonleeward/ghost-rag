Copy and export data
¶
Copy as MD
Use this guide to choose between moving data inside Tinybird and exporting query results to external systems.
To move data between Tinybird Data Sources, use
Copy Pipes
.
To export query results to external systems, use
Sinks
.
For destination-specific setup, see
Kafka Sink
,
S3 Sink
, and
GCS Sink
.
See
Copy and export data
for the full workflow.
