S3 Status Variables
A list of status variables for monitoring the S3 engine's interactions with the cloud service, although specific variables are not extensively documented here.

MariaDB 10.5.4
.
This page documents status variables related to the
S3 storage engine
. See
Server Status Variables
for a complete list of status variables that can be viewed with
SHOW STATUS
