uniqTheta functions
В¶
Copy as MD
UniqTheta functions perform set operations like union, intersection, and difference on two
uniqThetaSketch
objects. These functions return a new
uniqThetaSketch
object representing the result of the operation.
A
uniqThetaSketch
object is an approximate data structure used to estimate the number of unique elements in a set. It is typically created using the
uniqThetaState
aggregate function.
uniqThetaUnion
В¶
Combines two
uniqThetaSketch
objects to produce a new sketch representing the approximate union of their unique elements.
Syntax
В¶
uniqThetaUnion(uniqThetaSketch, uniqThetaSketch)
Arguments
В¶
uniqThetaSketch
:
uniqThetaSketch
. The first input sketch.
uniqThetaSketch
:
uniqThetaSketch
. The second input sketch.
Returns
В¶
A new
uniqThetaSketch
object containing the approximate union of the input sketches.
uniqThetaSketch
.
Example
В¶
SELECT
    finalizeAggregation(uniqThetaUnion(a, b)) AS a_union_b,
    finalizeAggregation(a) AS a_cardinality,
    finalizeAggregation(b) AS b_cardinality
FROM
(
    SELECT
        arrayReduce('uniqThetaState', [1, 2]) AS a,
        arrayReduce('uniqThetaState', [2, 3, 4]) AS b
)

uniqThetaNot
В¶
Determines the approximate set difference (elements in the first sketch but not in the second) between two
uniqThetaSketch
objects, yielding a new sketch.
Syntax
В¶
uniqThetaNot(uniqThetaSketch, uniqThetaSketch)
Arguments
В¶
uniqThetaSketch
:
uniqThetaSketch
. The first input sketch (minuend).
uniqThetaSketch
:
uniqThetaSketch
. The second input sketch (subtrahend).
Returns
В¶
A new
uniqThetaSketch
object containing the approximate difference of the input sketches.
uniqThetaSketch
.
Example
В¶
SELECT
    finalizeAggregation(uniqThetaNot(a, b)) AS a_not_b,
    finalizeAggregation(a) AS a_cardinality,
    finalizeAggregation(b) AS b_cardinality
FROM
(
    SELECT
        arrayReduce('uniqThetaState', [2, 3, 4]) AS a,
        arrayReduce('uniqThetaState', [1, 2]) AS b
)
