Free Virtual Training: From Kafka to Analytics April 2024
The ideal course for data engineers and developers to extend their Apache Kafka and Confluent skills. Learn how to leverage your streaming data to build real-time analytics applications.
This course is intended for data engineers and developers working with Apache Kafka, either self-hosted or managed. In this 90-minute free virtual course, you’ll learn how to:
Unify multiple batch and streaming data sources
Build data analysis pipelines using SQL
Use Materialized Views for performance
Connect your data to user-facing applications
