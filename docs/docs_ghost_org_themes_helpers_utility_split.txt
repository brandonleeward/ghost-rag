Utility
split
Usage:
{{#split "apple-banana-pear" separator="-"}}
The
{{#split}}
helper is designed to split a string into separate strings.  It can be used in block or inline mode.
The
{{#split}}
helper returns an array, suitable for iteration with
{{#foreach}}
, with individual elements of the array suitable for any helper that expects a string.
Individual elements of the array may be addressed as
{{this}}
within a
{{#foreach}}
loop.

{{split}}
is designed for strings. If it receives a non-string, it attempts to convert it to a string first.
​
The separator attribute
By default, strings are split at each ”,”. The
separator=""
attribute allows settings the split location to an arbitrary value.
Passing an empty string for the separator results in splitting to single characters.
Separators may be multiple characters.
