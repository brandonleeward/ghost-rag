Utility Helpers
Utility helpers are used to perform minor, optional tasks. Use this reference list to discover what each handlebars helper can do when building a custom Ghost theme.
Tag
Description
asset
Outputs cachable and cache-busting relative URLs to various asset types
block
Used along with
{{contentFor}}
to pass data up and down the template hierarchy
body_class
Outputs dynamic CSS classes intended for the
<body>
tag
color_to_rgba
Converts a color value into an RGBA string
concat
Concatenate and link multiple things together
contrast_text_color
Returns a readable text color for a given background
encode
Encode text to be safely used in a URL
ghost_head
/
ghost_foot

link_class
Add dynamic classes depending on the currently viewed page
log
In development mode, output data in the console
json
Safely serialize values for inline JSON output
pagination
Helper which outputs formatted HTML for pagination links
partials
Include chunks of reusable template code
plural
Output different text based on a given input

Outputs classes intended for your post container
prev_post

post

reading_time
