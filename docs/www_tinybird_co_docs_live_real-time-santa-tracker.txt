Building a Real-Time Santa Tracker
Getting Santa from house to house is quite a logistics problem! Will the elves finish building the toys in time? Who's on the naughty and nice lists? What's Santa's most efficient route? Join us for a fun, festive live coding session where we'll build a live real-time Santa Tracker using streaming data from Santa's sleigh, CDC data from the workshop, and naughty and nice lists stored in an S3 bucket.
In this live coding session, you'll learn how to...
Solve complex inventory and logistics problems with real-time data
Build a real-time data pipeline using simple HTTP requests and SQL
Enrich event streams with data on files in object storage
Capture change data from an application database for real-time analytics
Create scalable, low-latency APIs to empower your team to build things with data
