Building a complete IoT backend with Redpanda and Tinybird
Come learn how to build a complete IoT backend in 45 minutes using Redpanda and Tinybird. We'll set up a Redpanda Cloud cluster and stream IoT data to Tinybird for exploration, transformation, and publication.
In this live coding session, we will:
Spin up a Redpanda Cloud cluster to start streaming IoT data collected from Tour de France cyclists.
Connect Redpanda to Tinybird to enrich data streams and analyze the data for trends and insights about rider elevation and pace.
Use Tinybird Pipes to transform IoT data streams into something consumable and actionable.
Publish Tinybird Pipes as API endpoints in a single click and demonstrate how a frontend can use them.
