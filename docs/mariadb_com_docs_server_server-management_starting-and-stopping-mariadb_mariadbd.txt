mariadbd
Describes the `mariadbd` binary (formerly `mysqld`), which is the core database server executable.
Previously, the client was called
mysqld
. It can still be accessed under this name, via a symlink in Linux, or an alternate binary in Windows.
See
mariadbd-options
for details.
